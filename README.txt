- Source code được chạy trên Visual Studio Code
- Các bước chạy source code: 
 + Mở source code trên Visual Studio Code
 + Mở Terminal trên thanh công cụ
 + Chạy lệnh npm install để cài đặt Node module
 + Sau khi đã cài xong, chạy lệnh ng serve --o để chạy source code, trang web sẽ tự động mở 
 (nếu trình duyệt không tụ mở thì truy cập http://localhost:4200/ để mở website)
